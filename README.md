## RED Social

Objetivo
Desarrollar una aplicación de red social que permita a los usuarios registrarse, iniciar sesión, publicar
mensajes, ver el muro, buscar mensajes, gestionar su perfil y realizar acciones adicionales.

## .env
URI_MONGO=
JWT_SECRET=
JWT_REFRESH=
MODO=developer
